# OrangeHRM Cypress Automation

## Project Overview

This project demonstrates automated end-to-end testing of the OrangeHRM web application using Cypress and TypeScript. The test suite covers core business workflows to ensure application functionality, reliability, and regression stability.

## Test Scenarios

The automated test suite includes:

- User Login
- User Logout
- Dashboard Validation
- Employee Creation
- Employee Search
- Employee Information Validation
- Employee Deletion
- Form Validation
- Navigation Between Modules
- Positive and Negative Test Scenarios
- End-to-End User Workflows

## Technologies Used

- Cypress
- TypeScript
- Node.js
- npm

## Testing Approach

- End-to-End (E2E) Testing
- Functional Testing
- Regression Testing
- UI Testing
- Automated Test Execution

## Project Structure

- Test Scripts
- Page Objects (if applicable)
- Fixtures
- Custom Commands
- Configuration Files

## How to Run

```bash
npm install
npx cypress open
```

or

```bash
npx cypress run
```

## Purpose

The purpose of this project is to demonstrate practical experience in building and maintaining automated UI tests using Cypress while applying software testing best practices and improving test reliability.
